**🛒 BuyVia Vanilla**

BuyVia is a lightweight e-commerce front-end application built with vanilla JavaScript, HTML, and CSS, designed to simulate a real online shopping experience. It focuses on core frontend logic such as product rendering, cart state management, and checkout flow without relying on frameworks.

🔗 Live Demo

**Deployed on [Netlify](https://buyvia.netlify.app/)**

**🧰 Tech Stack**

  - HTML5
  - CSS3
  - JavaScript (ES6+)
  - Vite (development & production build tool)
  
  
**✨ Key Features**

  - Dynamic product listing
  - Add, update, and remove items from cart
  - Persistent cart state using local storage
  - Multi-page structure (home, product, cart, checkout, profile)
  - Responsive layout for mobile and desktop
  - Production-ready build setup using Vite
